# Screenshots for README

This folder should contain the following screenshots for the README:

## Required Screenshots:

1. **dashboard.png** - Main dashboard interface showing:
   - Upload section with drag & drop
   - Brand header with eye icon
   - Empty state before upload

2. **prediction-results.png** - Results page showing:
   - Uploaded image preview
   - KPI cards (Predicted Action, Confidence, Total Classes)
   - Prediction distribution chart (horizontal bar chart)
   - Confidence gauge (semi-circle chart)
   - "Analyze Another Image" button

## How to Create Screenshots:

1. Start the backend: `cd backend && python app.py`
2. Open frontend in browser: `http://localhost:8080`
3. Take screenshot of empty dashboard → save as `dashboard.png`
4. Upload an image and get results → take screenshot → save as `prediction-results.png`

## Tips:
- Use high resolution (1920x1080 or higher)
- Include the full browser window
- Make sure text is readable
- Use PNG format for better quality
